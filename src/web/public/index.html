RrratCoder - AI Coding Agent
Set your DeepSeek API Key in the top bar, describe your task, and the agent will generate code, run validation, and auto-correct errors. Click "API Reference" to see available endpoints.

API Endpoints

GET
/api/health
Server health check with uptime
curl http://localhost:3000/api/health
GET
/api/info
Project info: name, version, modules, test stats
curl http://localhost:3000/api/info
GET
/api/key/status
Check if DeepSeek API Key is configured
curl http://localhost:3000/api/key/status
POST
/api/key/set
Store API Key (AES-256-GCM encrypted)
curl -X POST http://localhost:3000/api/key/set -H "Content-Type: application/json" -d '{"password":"mypass","apiKey":"sk-..."}'
POST
/api/key/clear
Delete stored credentials
curl -X POST http://localhost:3000/api/key/clear
POST
/api/run
Execute a coding task with the agent
curl -X POST http://localhost:3000/api/run -H "Content-Type: application/json" -d '{"task":"Create hello.ts","password":"mypass"}'
